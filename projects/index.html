Japanese Language Processing in Python

Exploring Language

I am currently studying NLP one-on-one with a professor using the online textbook Natural Language Processing with Python. I hope to use the Tool Kit and the techniques learned in this book to create a Japanese notecard webapp in the near future. This webapp will be for those studying Japanese to upload movie subtitle files into and the program will automatically tokenize the individual words, conjugate verbs and adjectives to their dictionary forms, and then create flash cards to be studied so the user will be able to watch the movie with understanding. I will write the code using mostly modules that I create on my own to continue my exploration of NLP.

Future Features of the WebApp

  • Tokenize without the help of whitespace
  • Conjugate verbs and adjectives back to dictionary form
  • Colloquial speech recognition (slang/若者言葉)
  • Parse POS taggers and integrate the sentence in which the word is used in context to help with studying
  • Contain a database of previously studied words for each user so only new vocabulary will be studied
