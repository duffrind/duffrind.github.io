My current coursework is:

Japanese NLP Thesis:

I am researching which features influence the pronunciation used for Kanji in different context by using Bayesian Networks. I will then use this data to help Japanese learners guess which pronunciation will be used for new vocabulary that he or she has not yet seen before.

3D Data Visualizations:

This is a tutorial that I created as an extension of last semester's Data Visualization course. We are using Unity and Vuforia to create visualizations for crime statistics.

Statistical Inference:

This course starts with reviewing Descriptive Statistics and builds off of them to teach the statistical models that are used in Machine Learning.

Computer Architecture and Digital Hardware:

We build a simulated computer from the ground up starting with NAND gates. This course follows Nand2Tetris.

Computer Science Seminar:

Everyone takes turns presenting an interesting research paper or other topic to the class.
